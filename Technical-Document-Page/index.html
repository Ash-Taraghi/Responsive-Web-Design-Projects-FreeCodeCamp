Scope

There are 2 types of scope to discuss in pyton, which is local and global scopes.

Local scopes are automatically given to variables declared within a function as that variable exists only in that function to the program. Global variables are either declared outside of the function or you can use the "global" keyword when assinging a new variable in a function to force that variable to have a global scope even though it is assinged in a local enviroment.

global x = 5
Functions

Functions are blocks of code that only execute when they are called. You can pass data amongst functions known as parameters, into the function. Functions have the ability to return data as their result.

In python, functions are declared using the def keyword and all the code inside the function must be indented. Parameters being passed in must be declared in the brackets of the function and if the function is supposed to return a value a return statement is required.
